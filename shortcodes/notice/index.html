    notice

    The notice shortcode shows 4 types of disclaimers to help you structure your page.

    Note

    {{% notice note %}}
A notice disclaimer
{{% /notice %}}

    renders as

    A notice disclaimer

    Info

    {{% notice info %}}
An information disclaimer
{{% /notice %}}

    renders as

    An information disclaimer

    Tip

    {{% notice tip %}}
A tip disclaimer
{{% /notice %}}

    renders as

    A tip disclaimer

    Warning

    {{% notice warning %}}
An warning disclaimer
{{% /notice %}}

    renders as

    An warning disclaimer
