With sub-graphs and some style

Show code...

{{}} graph LR; X --> Y linkStyle 0 stroke:#f00,stroke-width:4px; Y --> Z Z --> X linkStyle 1,2 interpolate basis stroke:#0f0,stroke-width:2px; X --> A1 subgraph right A2 --> B2 B2 --> C2 end subgraph left A1 --> B1 B1 --> C1 end C1 --> X Z --> A2 C2 --> Z

    style Y fill:#f9f,stroke:#333,stroke-width:4px

    classDef left fill:#ccf,stroke:#f66,stroke-width:2px,stroke-dasharray: 5, 5
    class A1,B1,C1 left
{{< /mermaid >}}
